# New Case Study - Checklist

To add a case study to your site, send me whatever you have from the list below.
Only the first item is required — the more you can fill in, the richer the case study looks.

## Required

- [ ] **Client / organization name**

## Optional (include whatever you have)

- [ ] **Logo** — transparent background, as PNG, SVG, or WebP
- [ ] Is the logo **dark** or **light** colored?
- [ ] **Project timeline** — e.g. "March 2024 – Present"
- [ ] **Description** — a sentence or two on what the project was about
- [ ] **Key contributions** — a few bullet points on what was delivered
- [ ] Which **service** this work falls under
- [ ] **Testimonial** (skip if there's no quote) — if you have one, I need all of these together:
  - [ ] The **quote**
  - [ ] **Who said it** — full name
  - [ ] Their **company or organization**
  - [ ] Their **job title / role** _(if they have one)_
